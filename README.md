# Quiz Website von Sverre und Jonas

## Welche Themen gibt es?
Es gibt Quizze und Infoseiten zu HTML, CSS, JavaSkript und den Aufbau von PCs.

## Quizze
Zu jedem der oben gennanten Themen gibt es ein Quiz mit jeweils 10 Fragen. Damit kannst du dein Wissen, welches du im Infotext erlernt hast, überprüfen.
Ein iFrame unter der Frage zeigt dir, wenn du auf eine Antwort klickst, ob die Antwort richtig oder falsch ist.

## Technologien
Die Website wurde mit **HTML** und **CSS** programmiert und mit **Netlify** gehostet. Der Code ist auf **GitHub** einsehbar.

## Wie ist dieses Projekt entstanden?
Das Projekt ist im Rahmen der Projekt Arbeit von Sverre und Jonas entstanden. Die Aufgabe war es, die erste eigene Website zu programmieren. Dabei gab es bestimmte Vorgaben, wie z.B. das die Website offline funktionieren muss, oder dass sie anders aussehen muss, jenachdem, ob das Gerät im Hoch- oder Querformat ist.
