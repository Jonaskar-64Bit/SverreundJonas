CSS in HTML – Die wichtigsten Grundlagen

Was ist CSS?

CSS steht für Cascading Style Sheets und wird verwendet, um das Aussehen von Webseiten zu gestalten. Während HTML die Struktur und den Inhalt einer Seite festlegt, übernimmt CSS das Design. Mit CSS können Farben, Schriftarten, Abstände, Größen und Layouts definiert werden.

Wie funktioniert CSS?

CSS arbeitet mit sogenannten Selektoren. Diese bestimmen, welche HTML-Elemente gestaltet werden sollen. Ein CSS-Befehl besteht aus einem Selektor sowie einer oder mehreren Eigenschaften mit entsprechenden Werten. Dadurch weiß der Browser, wie ein bestimmtes Element dargestellt werden soll.

Die Einbindung von CSS in HTML

CSS kann auf drei verschiedene Arten in eine HTML-Seite eingebunden werden. Beim Inline-CSS wird der Stil direkt in ein HTML-Element geschrieben. Internes CSS befindet sich im <style>-Bereich im <head> der Seite. Am häufigsten wird jedoch externes CSS verwendet. Dabei liegt der Code in einer separaten .css-Datei, die mit dem <link>-Tag eingebunden wird.

CSS-Regeln und Eigenschaften

Eine CSS-Regel besteht aus einem Selektor und einem Deklarationsblock. Typische Eigenschaften sind zum Beispiel color für die Textfarbe, font-size für die Schriftgröße oder margin für Abstände. Diese Eigenschaften ermöglichen eine präzise Gestaltung einzelner Elemente oder ganzer Webseitenbereiche.

Die Trennung von Inhalt und Design

Ein großer Vorteil von CSS ist die klare Trennung von Struktur und Gestaltung. Während HTML den Inhalt beschreibt, kümmert sich CSS ausschließlich um das Layout. Dadurch bleibt der Code übersichtlich und das Design kann zentral für mehrere Seiten gleichzeitig geändert werden.

Interaktive Effekte mit CSS

Mit CSS lassen sich auch einfache Interaktionen umsetzen. Ein Beispiel ist der :hover-Pseudoselektor. Dieser sorgt dafür, dass sich das Aussehen eines Elements verändert, sobald man mit der Maus darüberfährt – etwa indem sich die Hintergrundfarbe eines Buttons blau färbt.
